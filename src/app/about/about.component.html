Technology Feature Description
Azure Web App + API App
  • Created a Web App to host the Angular application
  • Created an API App to host the C# API application
  • Created a resource group and a service plan to keep these components together
Continuous Integration (CD/CI)
  • Connected to the GIT repository
  • Automatically pulls the latest code after every commit
Continuous Deploy (CD/CI)
  • Build and Deploy using Kudu Script
  • Install NPM packages on the Azure server
  • Create a production build in the repository/dist folder on Azure
  • Deploy files from repository/dist to the deployments folder on Azure
Testing Jasmine Unit Tests
  • Created and modified the Jasmine unit tests
  • Added dependency modules
  • Tested HTML elements and variable values
  • Tested function inputs and outputs
  • Command to run tests: ng test
Protractor End-to-End Tests
  • Created and modified the Protractor tests
  • Tested sequential actions and checked the values along the way
  • Command to run tests: ng e2e
.NET Core RESTful Web API
  • Created a .NET Core application using Visual Studio 2017
  • Created an API to get the user input and return the result
  • Implemented GET, POST and DELETE methods
  • Implemented attribute routing
CORS - Cross Origin Resource Sharing
  • Allowed requests from other domains
  • Added Access-Control-Allow-Origin in the customHeader in the Web.config file
Styling Responsive Web Design
  • Used @media queries to add screen breakpoints
  • Reordered the header components for a small screen
  • Resized the boxes for a small screen
Flex Box
  • Used flex box to align multiple divs in a row
  • Added justify-content to manage the spacing of the elements
SCSS
  • Created a global variables.scss file to store custom colors and styles
  • Used @import to import the variables in the required components
  • Used nesting for styling elements within a context like link:hover, tables
Performance Compress Images
  • The images are resized and then compressed using resizeimage.net
  • Image file compressed from 139 kb to 15 kb
Webpack
  • Compiling and transpiling TypeScript and SCSS
  • Bundling and minifying JavaScript and CSS files
  • Creating production builds
Search Engine Optimization (SEO)
  • Added a title and description Open Graph meta tag
  • Added a thumbnail image for link sharing
Angular Routing
  • Created an app-routing module
  • Added default route, page not found route
Services
  • Created an HTTP service that makes calls to the .NET Core RESTful Web API
  • Service returns an Observable to the calling function
Components
  • The app is modularized into small components for easy maintenance
  • Some components are reused from previous projects
